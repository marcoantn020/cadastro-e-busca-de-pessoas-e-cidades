
# Frontend sistema de cadastro de pessoas e cidades
  - Material UI
  - axios
  - react-router-dom v6
  - yup
  - json-server

#### Comands 
  ```
    npm install
    ou
    yarn install
  ```
#### Run frontend
  ```
    yarn start
    ou 
    npm run start
  ```
#### Run mock backend
  ```
    yarn server
    ou 
    npm run server
  ```
